# Dre's Grind 💪

A personal fitness and lifestyle accountability dashboard built for 
the grind. Tracks workouts, weight, nutrition, and the journey to 
San Diego.

**Live app:** https://dretheprophet-pixel.github.io/dres-grind

---

## What It Does

### Today
Daily dashboard with a San Diego wedding countdown (May 23, 2026), 
workout streak, no-drink streak, Whole30 day counter, meal check-in, 
and weight snapshot. Everything you need to see at a glance.

### Workouts
Log sessions from a 10-activity dropdown — Strength, Core, Boxing, 
Kickboxing, HIIT, Pilates, Mindful Cooldown, Outdoor Walk, Run, 
Basketball. Track duration, effort (1–5), energy level, and calories.

### Nutrition
Whole30 tracker with daily meal logging, per-meal compliance 
toggling, compliance streak, meal history, and a Whole30 quick 
reference (Yes / No / Gray Area).

### Meal Prep
Recipe library with ingredients and instructions, Whole30 tagging, 
a weekly meal planner grid, and a shopping list generator that 
consolidates ingredients from your planned meals.

### Weight
Weekly check-ins tracked toward a 205 lb goal. Milestone markers 
at 220, 215, 210, and 205 lbs. San Diego pace line shows whether 
you're ahead or behind the linear track to hit goal by May 23rd.

### No-Drink Streak
Streak counter with milestone celebrations at 7, 14, 21, 30, 60, 
90, and 365 days. Tracks all-time best streak and full streak history.

### Weekly & Monthly Stats
Charts, heatmaps, and stats for both workout consistency and 
nutrition compliance. Dual heatmaps on the monthly view — workout 
days and nutrition compliance side by side.

---

## Tech Stack

- Vanilla HTML, CSS, JavaScript — no frameworks, no build process
- Chart.js for all data visualizations
- localStorage for data persistence
- Google Fonts — Bebas Neue, Barlow, Barlow Condensed
- Single file architecture (`index.html`)

---

## Running Locally

No setup needed. Clone the repo and open `index.html` in any browser:
```bash
git clone https://github.com/dretheprophet-pixel/dres-grind.git
cd dres-grind
open index.html
```
