# Dre's Grind — Project Context

## What this is
A personal fitness and lifestyle accountability dashboard. Single HTML file, 
vanilla JS, Chart.js for charts, localStorage for data persistence.

## Redesign Goals (v2)
- Fresh, clean, lighter aesthetic — premium wellness app feel
- Whole30 nutrition tracking with daily compliance logging
- No-drink streak tracker
- Meal prep hub with shareable recipes and shopping lists
- Milestone countdown — San Diego wedding, May 23rd 2026
- Workout activity dropdown (replacing checkboxes)
- Weight tracking toward goal weight
- Weekly and monthly stats with charts and heatmaps

## Code Preferences
- Single HTML file (no build process, no frameworks)
- Explain what you're building before writing it
- Keep changes focused — one feature at a time
- Commit after each major feature is working

## Branch
Always work on `redesign/v2` — never commit directly to main
